⚙️ 🛠 INSTALLATION

1️⃣ Clone

git clone https://github.com/YOUR_USERNAME/agrisat.git
cd agrisat

2️⃣ Frontend

cd frontend
npm install
npm run dev

3️⃣ Backend

cd backend
pip install -r requirements.txt
uvicorn main:app --reload

4️⃣ Environment

Frontend:

VITE_API_URL=your_backend_url
VITE_GOOGLE_CLIENT_ID=your_google_client_id

Backend:

GNEWS_API_KEY=your_key
WEATHER_API_KEY=your_key

---

🚀 DEPLOYMENT

Frontend → Vercel
Backend → Render / Cloud Run
